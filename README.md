# dopplereffekt
======

### Requirements
- npm (version > 1.2.10)
- Grunt (`npm install -g grunt-cli`)
- Sass (version > 3.3 http://sass-lang.com/install)
- Bower

### Install (go to the project repertory with a terminal and type:)
- `npm install`
- `bower install`
- `grunt serve`
- look your browser

### How to release
- do your changes
- `grunt serve:dist` (check if everything is OK and ready to deploy)
- `grunt deploy`
